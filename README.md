# ATM-SIMULATION-SYSTEM
# 🏧 ATM System — C++ OOP Project

A console-based ATM simulation system built with **C++** using
**Object-Oriented Programming** concepts as a university project.

---

## ✨ Features

- 🔐 **PIN Authentication** — User must enter a correct PIN to access the account
- 💰 **Check Balance** — View current account balance at any time
- 💵 **Deposit Money** — Add money to the account instantly
- 🏧 **Withdraw Cash** — Withdraw amount with insufficient funds check
- 📋 **Transaction History** — View a list of recent transactions
- 🔒 **Account Lockout** — Account locks after 3 incorrect PIN attempts
- ✅ **Input Validation** — Invalid inputs are handled without crashing
- 🖥️ **Menu Driven** — Easy to navigate console menu interface

---

## 🎯 Objectives

1. **Simulate a Real ATM** — Build a working ATM system that replicates basic real-world banking functions
2. **Apply OOP Concepts** — Use classes, objects, constructors, and methods to organize the code properly
3. **Implement Authentication** — Create a secure PIN-based login system with account lockout on failed attempts
4. **Handle Transactions** — Correctly process deposit and withdrawal operations with balance validation
5. **Validate User Input** — Ensure all inputs are checked and handled before any operation is performed
6. **Display Clear Output** — Show account details, transaction results, and messages in a readable format
7. **Practice Programming Skills** — Apply C++ knowledge learned in class to build a real and functional project
